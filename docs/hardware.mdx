# Hardware

* **Location:** Rotterdam (europe)
* **CPU:** CPU: AMD Threadripper Pro 7965WX
* **RAM**: 512GB
* **Network interface:** 10Gbps

### 2. Backup Server #1

* **Location:** Madrid (europe)
* **CPU:** CPU: AMD EPYC 9254
* **RAM**: 384GB
* **Network interface:** 10Gbps

### 3. Backup Server #2

* **Location:** Tokyo (asia-pacific)
* **CPU:** AMD EPYC 7443P 
* **RAM**: 256GB
* **Network interface:** 10Gbps

* **Location:** Poland (europe)
* **CPU:** CPU: AMD Ryzen 7950X
* **RAM**: 192GB

Our commitment to high-specification hardware and an 'always evolving' geographical footprint minimizes vote latency whilst maximising redunadncy.
